# GuardSense

Esse projeto trata-se de um sistema de segurança doméstica e comercial com um App junto para monitoramento a partir do aplicativo e também para controlar o sistema físico.

- O App é feito com Kotlin
- A parte física do projeto é desenvolvida com C++
